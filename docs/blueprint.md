# **App Name**: MoTrack

## Core Features:

- Order Management: Customer order creation and management (CRUD operations).
- Role-Based Access Control: User roles management: admin, employee, and installer. Permissions control what each role can access and do.
- Milestone Tracking: Visual tracking of project milestones based on order type (delivery, stitching, stitching+installation).
- Installer Mobile View: Mobile app view for installers: Task assignments, progress updates (including time stamps).
- Customer Order Tracking: Public-facing page for customers to track order progress using a tracking code.
- Detailed Order History: Display complete details for an order. Allow admins to filter orders by employee/installer, salesperson, and date range.
- AI-Powered Dispatcher Assistant: AI tool that analyzes current workloads of installers, delivery locations, and deadline projections. Generates an installation schedule for the dispatcher's approval.

## Style Guidelines:

- Primary color: Blue (#4285F4) for trust and reliability, reflecting the app's tracking functionality.
- Background color: Light gray (#F5F5FA), a desaturated variant of blue to ensure focus remains on data presentation.
- Accent color: Green (#34A853) used for 'completed' milestones, task confirmations, and successful deliveries.
- Body and headline font: 'Inter' sans-serif, with a modern, objective look. Suitable for UI elements and tabular data.
- Icons representing different milestones and order types. Standard iconography from a library like FontAwesome, clearly representing actions/status.
- Clean, card-based layouts for order and task information, using white space to improve readability.
- Subtle transitions for status updates and task assignments. Progress bars provide visual feedback on order progression.
